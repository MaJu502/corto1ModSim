# transformacin inversa (ej1)
def ej1Inversa(elementos,temp):
    X = np.zeros(elementos)
    for j in range(elementos):
        U = np.random.uniform()
        x = 0
        s = np.exp(-temp)

        while U > s:
            x += 1
            s += np.exp(-temp) * (temp ** x) / np.math.factorial(x)

        X[j] = x

    return X

# Mtodo de rechazo usando la distribucin uniforme como propuesta
def Metodo_De_Rechazo():
    while True:
        X = np.random.randint(0, max_x)  # Generar un nmero aleatorio de la distribucin uniforme
        U = np.random.uniform(0, 1)  # Generar un nmero aleatorio U entre 0 y 1
        if U <= poisson.pmf(X, lambda_v) / C:
            return X

2. Discuta las ventajas y desventajas de cada mtodo en trminos de eficiencia y precisin.

El mtodo de transformacin inversa tiene la ventaja de la exactitud, ya que genera muestras directamente de la distribucin de Poisson, entonces genera valores mas acertados, sin mencionar que su implementacin es relativamente sencilla. Por otra parte, una desventaja potencial de este mtodo es su eficiencia, pues en algunos casos puede requerir clculos adicionales.

En cuanto al mtodo de rechazo, su mayor ventaja radica en su generalidad ya que puede ser aplicado para muestrear cualquier distribucin. Adems, la simplicidad de utilizar la distribucin uniforme simplifica el proceso y no requiere clculos demasiado complejos. Por otra parte la eficiencia puede ser una desventaja si la eleccin de la constante C no es la mejor, o si la distribucin propuesta no se ajusta bien a la distribucin deseada. En estos casos, pueden haber muchos rechazos, lo que resulta en un algoritmo ms lento.
